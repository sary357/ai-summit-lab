# ai-summit-lab
- For AI summit held at 2024-12-11

# Prerequisite
## Poetry

```
$ curl -sSL https://install.python-poetry.org | python3 -
```
## Jupyter notebook
```
$ poetry add jupyter
```

# References
## Use poetry shell
```
$ poetry shell
```

## Launch Jupyter
```
$ jupyter notebook --ip 0.0.0.0
```
